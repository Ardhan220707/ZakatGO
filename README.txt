**ZakatGO** adalah sebuah aplikasi web kalkulator Zakat Maal sederhana yang dirancang untuk membantu pengguna menghitung kewajiban zakat mereka dengan cepat, mudah, dan akurat. Aplikasi ini menekankan kemudahan penggunaan serta transparansi dalam setiap proses perhitungannya, sehingga pengguna dapat memahami dengan jelas dasar dari setiap hasil yang ditampilkan.


Aplikasi ini telah di-deploy secara publik dan dapat diakses langsung melalui GitHub Pages di tautan berikut: **[https://ardhan220707.github.io/ZakatGO/](https://ardhan220707.github.io/ZakatGO/)**. Dengan begitu, siapa pun dapat menggunakan kalkulator zakat ini tanpa perlu mengunduh atau menginstal apa pun di perangkat mereka.

Bagi pengguna yang ingin menjalankan aplikasi ini secara lokal di komputer, langkah-langkahnya cukup sederhana. Pertama, lakukan *clone* repositori melalui perintah berikut di terminal atau command prompt:

======================================================
git clone https://github.com/Ardhan220707/ZakatGO.git
======================================================

Setelah itu, masuk ke direktori proyek dengan perintah:

=======================================================
cd ZakatGO
=======================================================

Kemudian, buka file `index.html` menggunakan browser favorit Anda seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge untuk langsung menjalankan aplikasi.

ZakatGO dikembangkan sepenuhnya menggunakan teknologi *frontend* modern tanpa memerlukan *backend*. Struktur aplikasi ini dibangun menggunakan **HTML5** untuk kerangka dasar halaman web, **CSS3** untuk pengaturan tampilan seperti warna, font, dan gaya visual lainnya, serta **Bootstrap 5** sebagai *framework* utama untuk menciptakan antarmuka yang rapi, responsif, dan mudah diakses di berbagai perangkat. Sementara itu, logika utama aplikasi dijalankan menggunakan **JavaScript (ES6+)**, yang berfungsi untuk mengelola input pengguna, memvalidasi data, menghitung nilai zakat berdasarkan nisab, serta menampilkan hasil perhitungan secara dinamis dan interaktif.

Secara keseluruhan, ZakatGO merupakan proyek sederhana namun bermanfaat yang memanfaatkan kekuatan teknologi web untuk membantu masyarakat menjalankan kewajiban zakat dengan lebih mudah dan transparan.
